Dirbuster







Why is it useful?


Just like credentials are sometimes found within the source page of a website, there are often files and directories left accessible. Sometimes a developer will add an admin page as a quick bypass to edit the website but forget to secure it. Any information found in these are a potential vulnerability to the website.



What is Dirbuster?

Dirbuster is a web application tool designed to brute force directory and file names. In other words, this tool tries every variation of possible extensions for a website from a chosen wordlist.


This tool is multi-threaded and works over http and https, and includes custom file extensions to try.
