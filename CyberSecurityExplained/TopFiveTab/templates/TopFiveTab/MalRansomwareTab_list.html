What is Malware?

Malware is a huge problem today. Thousands of malware programs are designed each day to steal your data and ransomware is costing companies billions of dollars each year. Malware is malicious in design causing damage to data systems or gaining unauthorized access to a network. Malware creators have one goal and that is to make money illegally. Types of Malware software can be broken up into two categories. The first category shows the way malware software is spread. The second category explains how malware is harmful.

The way malware is spread.
  • Viruses
    • -Malicious software that self-replicates into multiple files on your computer. When you run these files, the virus is ran as well.
  • Worm
    • -Software that uses a computer network to spread to many computers by self-replication. Worm code can spread without host programs or human assistance.
  • Trojan
    • -Deceiving malicious software that hides from the user. Trojan code can hide as a legitimate program that will infect your computer. A fake antivirus program is an example of a trojan program.

How malware does harm.
  • Spyware
    • -Software that gains access to your computer and can monitor what you search for on the web or what you type on your keyboard. This information is given to other parties.
  • Adware
    • -Advertising-supported software that shows the users advertising generating revenue for creator.
  • Scareware
    • -This software shows the user content that often scares or tricks the user into thinking they must pay. A creator can pretend to be the authorities and demand pay or else the user could get arrested.
  • Ransomware
    • -The creator of this software locks the users computer down using encryption. They then hold important data ransom until the user pays the creator money.
  • Rootkits
    • -This software allows the creator access to otherwise restricted areas in the computer. Creators can access administrator-level control.
