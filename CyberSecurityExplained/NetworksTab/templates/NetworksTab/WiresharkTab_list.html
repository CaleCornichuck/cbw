What is Wireshark?

Wireshark is a network packet capture tool that captures network packets. These packets are shown on the granular scale and are used for offline analysis. This allows the user to look at the network up close and dive into the network. When using Wireshark the user can capture, interpret, filter, and inspect packets. Wireshark is used for troubleshooting networks with performance issues.



How Wireshark works.

  • Packet capture

    • Wireshark watches the network in real time grabbing streams of network (thousands of packets at a time).

  • Filtering

    • Wireshark filters through these streams and obtains specified information.

  • Visualization

    • Wireshark allows the users to dive into the middle of a network packet and visualize an entire network stream.

How can Wireshark help?

Wireshark can help find out if a router isn’t working correctly. If the user’s computer is slow Wireshark can dive into the packets and troubleshoot to find the issue. Wireshark can help with finding malicious activity, dropped packets, and latency issues.
