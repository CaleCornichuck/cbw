Network Security Attacks

A network security attack is when any means are used by the attacker to maliciously compromise network security.

An active attack is when the attack is highly malicious. The attacker can change data in route or destroy data. The attacker is detected by the users and is aggressive. The user can be locked out of the system.

A passive attack is a network attack that is undetected. The attacked will look at information and collect data.



Computer Network

A computer network is two or more computers that are connected to communicate data electronically. An example of a computer network is the internet.

Network Security

Network security is processes, polices, and practices that are designed to protect the confidentially, integrity, and an accessibility of computer networks and data using both software and hardware technologies.

One way the attacker can gain access is by guessing the password. Each password should be strong and complex to avoid an attack.









Network Security Components
  • Hardware

    • Hardware devices can be set up as out of the path of the networks traffic which is separate from the networks traffic. This device then monitors traffic and alters users when they detect malicious threat. The hardware device can also be set up as in the path of networks traffic and directly blocks data the moment threats are detected. Firewall hardware blocks unwanted traffic from entering the systems in the first place.

  • Software

    • Software components are installed on computers and protect from internet threats. This second line of defense helps provide added protection.

  • Cloud Services

    • Cloud services is like the in the path of the networks traffic. The networks traffic goes through the cloud provider and gets scanned for potential threats and when found they are blocked.
