Wi-Fi Pineapple

A Wi-Fi Pineapple is a portable device created by Hak5 to perform pen tests and audit network security. This device can help white hat hackers find vulnerabilities in the network what black hat attackers can exploit.

White hat hackers are the ethical hackers. The good guys.

Black hat attackers are the malicious hackers. The bad guys.

Pen tests (penetration testing) is a cyber security technique also known as ethical hacking.

How the Wi-Fi Pineapple works.

The Pineapple is between a network and a user’s computer. The device is looking through data that flows between the two. Information and data traveling back and forth from the network to computer are interpreted by the pineapple’s controller using Wi-Fi.

Wi-Fi Pineapple MitM attacks

The Wi-Fi Pineapple can be used to conduct a man in the middle (MitM) attack by using a rogue access point. The attacked intercepts and relays, without detection, messages between two users in direct communication.



Hacking!

The hackers can gain access to the user’s network and computer without detection. The hacker can eavesdrop without the user noticing. The hacker can remain connected even when the pineapple is physically removed using remote control through the internet. All sensitive or confidential information shared can now be seen by the hacker. The hacker can use any information they see for personal gain or malicious intent. Make sure when ever using a computer the Wi-Fi connection is the intended connection.
