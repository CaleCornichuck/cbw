John-the-Ripper

John-the-Ripper is password testing and breaking program. It uses the CPU to crack the code, instead of GPU. While GPU is usable with John-the-Ripper, it is not as efficient as it's CPU methods.


Modes of John-the-Ripper

Like Hashcat, John-the-Ripper has many different modes for different situations. Some examples are:

  • Single Crack Mode

    • This mode uses info from UNIX files, such as full names and usernames. This is good when the password uses easily available info in the username.


  • Wordlist Mode

    • Wordlist uses a text file, and iterates through each line as a password. This is very similar to the Dictionary Attack.


  • Incremental Mode

    • This is John-the-Ripper's version of a brute-force attack. While it is good at cracking passwords, if the password is too complex, the process may never end.
